# Elon Moonbase Countdown

A static website tracking Elon Musk's pledge to build a "self-growing city on the Moon" within 10 years (Target: February 2036).

## Live Site
Currently deployed at: **[https://elonmoonbasecountdown.pages.dev/](https://elonmoonbasecountdown.pages.dev/)**

## Project Structure
- `index.html`: Main entry point with the hero, countdown, and quotes.
- `style.css`: Custom "SpaceX-themed" styles and animations.
- `script.js`: Logic for the 10-year countdown timer.
- `moonbase_hero.png`: Assets.

## Deployment
This site is currently hosted on Cloudflare Pages.

To deploy manually:
1. `npm install -g wrangler`
2. `wrangler pages deploy .`
